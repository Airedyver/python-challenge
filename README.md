<h2 dir="auto">&#x000A;<a id="unit-3-assignment-py-me-up-charlie" class="anchor" href="#unit-3-assignment-py-me-up-charlie" aria-hidden="true"></a>Unit 3 | Assignment - Py Me Up, Charlie</h2>&#x000A;&#x000A;<h2 dir="auto">&#x000A;<a id="background" class="anchor" href="#background" aria-hidden="true"></a>Background</h2>&#x000A;&#x000A;<p dir="auto">Well... you've made it!</p>&#x000A;&#x000A;<p dir="auto">It's time to put away the Excel sheet and join the big leagues. Welcome to the world of programming with Python. In this homework assignment, you'll be using the concepts you've learned to complete <strong>2 of 4</strong> Python Challenges. Which ones you choose are completely your choice. In fact, feel encouraged to complete them all if you can muster the time.</p>&#x000A;&#x000A;<p dir="auto">Each of these challenges encompasses a real-world situation where your newfound Python scripting skills can come in handy. These challenges are far from easy so expect some hard work ahead!</p>&#x000A;&#x000A;<h3 dir="auto">&#x000A;<a id="before-you-begin" class="anchor" href="#before-you-begin" aria-hidden="true"></a>Before You Begin</h3>&#x000A;&#x000A;<ol dir="auto">&#x000A;<li><p>Create a new GitHub repo called <code>python-challenge</code>. Then, clone it to your computer.</p></li>&#x000A;<li><p>Inside your local git repository, create a directory for <strong>2</strong> of the <strong>4</strong> Python Challenges. Use folder names corresponding to the 2 challenges that you have chosen to complete: <strong>PyBank</strong>, <strong>PyPoll</strong>, <strong>PyBoss</strong>, or <strong>PyParagraph</strong>.</p></li>&#x000A;<li><p>Inside of each folder that you just created, add a new file called <code>main.py</code>. This will be the main script to run for each analysis.</p></li>&#x000A;<li><p>Push the above changes to GitHub.</p></li>&#x000A;</ol>&#x000A;&#x000A;<h2 dir="auto">&#x000A;<a id="option-1-pybank" class="anchor" href="#option-1-pybank" aria-hidden="true"></a>Option 1: PyBank</h2>&#x000A;&#x000A;<p dir="auto"></p><div class="image-container"><a class="no-attachment-icon" href="/GW-Coding-Boot-Camp/10-16-2017-GW-Arlington-Class-Repository-DATA/raw/master/Homework/03-Python/Instructions/Images/revenue-per-lead.jpg" target="_blank"><img src="/GW-Coding-Boot-Camp/10-16-2017-GW-Arlington-Class-Repository-DATA/raw/master/Homework/03-Python/Instructions/Images/revenue-per-lead.jpg" alt="Revenue"></a></div>&#x000A;&#x000A;<p dir="auto">In this challenge, you are tasked with creating a Python script for analyzing the financial records of your company. You will be given two sets of revenue data (<code>budget_data_1.csv</code> and <code>budget_data_2.csv</code>). Each dataset is composed of two columns: <code>Date</code> and <code>Revenue</code>. (Thankfully, your company has rather lax standards for accounting so the records are simple.)</p>&#x000A;&#x000A;<p dir="auto">Your task is to create a Python script that analyzes the records to calculate each of the following:</p>&#x000A;&#x000A;<ul dir="auto">&#x000A;<li><p>The total number of months included in the dataset</p></li>&#x000A;<li><p>The total amount of revenue gained over the entire period</p></li>&#x000A;<li><p>The average change in revenue between months over the entire period</p></li>&#x000A;<li><p>The greatest increase in revenue (date and amount) over the entire period</p></li>&#x000A;<li><p>The greatest decrease in revenue (date and amount) over the entire period</p></li>&#x000A;</ul>&#x000A;&#x000A;<p dir="auto">As an example, your analysis should look similar to the one below:</p>&#x000A;&#x000A;<pre class="code highlight js-syntax-highlight plaintext" v-pre="true"><code>Financial Analysis&#x000A;----------------------------&#x000A;Total Months: 25&#x000A;Total Revenue: $1241412&#x000A;Average Revenue Change: $216825&#x000A;Greatest Increase in Revenue: Sep-16 ($815531)&#x000A;Greatest Decrease in Revenue: Aug-12 ($-652794)&#x000A;</code></pre>&#x000A;&#x000A;<p dir="auto">Your final script must be able to handle any such similarly structured dataset in the future (your boss is going to give you more of these -- so your script has to work for the ones to come). In addition, your final script should both print the analysis to the terminal and export a text file with the results.</p>&#x000A;&#x000A;<h2 dir="auto">&#x000A;<a id="option-2-pypoll" class="anchor" href="#option-2-pypoll" aria-hidden="true"></a>Option 2: PyPoll</h2>&#x000A;&#x000A;<p dir="auto"></p><div class="image-container"><a class="no-attachment-icon" href="/GW-Coding-Boot-Camp/10-16-2017-GW-Arlington-Class-Repository-DATA/raw/master/Homework/03-Python/Instructions/Images/Vote_counting.jpg" target="_blank"><img src="/GW-Coding-Boot-Camp/10-16-2017-GW-Arlington-Class-Repository-DATA/raw/master/Homework/03-Python/Instructions/Images/Vote_counting.jpg" alt="Vote-Counting"></a></div>&#x000A;&#x000A;<p dir="auto">In this challenge, you are tasked with helping a small, rural town modernize its vote-counting process. (Up until now, Uncle Cleetus had been trustfully tallying them one-by-one, but unfortunately, his concentration isn't what it used to be.)</p>&#x000A;&#x000A;<p dir="auto">You will be given two sets of poll data (<code>election_data_1.csv</code> and <code>election_data_2.csv</code>). Each dataset is composed of three columns: <code>Voter ID</code>, <code>County</code>, and <code>Candidate</code>. Your task is to create a Python script that analyzes the votes and calculates each of the following:</p>&#x000A;&#x000A;<ul dir="auto">&#x000A;<li><p>The total number of votes cast</p></li>&#x000A;<li><p>A complete list of candidates who received votes</p></li>&#x000A;<li><p>The percentage of votes each candidate won</p></li>&#x000A;<li><p>The total number of votes each candidate won</p></li>&#x000A;<li><p>The winner of the election based on popular vote.</p></li>&#x000A;</ul>&#x000A;&#x000A;<p dir="auto">As an example, your analysis should look similar to the one below:</p>&#x000A;&#x000A;<pre class="code highlight js-syntax-highlight plaintext" v-pre="true"><code>Election Results&#x000A;-------------------------&#x000A;Total Votes: 620100&#x000A;-------------------------&#x000A;Rogers: 36.0% (223236)&#x000A;Gomez: 54.0% (334854)&#x000A;Brentwood: 4.0% (24804)&#x000A;Higgins: 6.0% (37206)&#x000A;-------------------------&#x000A;Winner: Gomez&#x000A;-------------------------&#x000A;</code></pre>&#x000A;&#x000A;<p dir="auto">Your final script must be able to handle any such similarly-structured dataset in the future (i.e you have zero intentions of living in this hillbilly town -- so your script needs to work without massive re-writes). In addition, your final script should both print the analysis to the terminal and export a text file with the results.</p>&#x000A;&#x000A;<h2 dir="auto">&#x000A;<a id="option-3-pyboss" class="anchor" href="#option-3-pyboss" aria-hidden="true"></a>Option 3: PyBoss</h2>&#x000A;&#x000A;<p dir="auto"></p><div class="image-container"><a class="no-attachment-icon" href="/GW-Coding-Boot-Camp/10-16-2017-GW-Arlington-Class-Repository-DATA/raw/master/Homework/03-Python/Instructions/Images/boss.jpg" target="_blank"><img src="/GW-Coding-Boot-Camp/10-16-2017-GW-Arlington-Class-Repository-DATA/raw/master/Homework/03-Python/Instructions/Images/boss.jpg" alt="Boss"></a></div>&#x000A;&#x000A;<p dir="auto">In this challenge, you get to be the <em>boss</em>. You oversee hundreds of employees across the country developing Tuna 2.0, a world-changing snack food based on canned tuna fish. Alas, being the boss isn't all fun, games, and self-adulation. The company recently decided to purchase a new HR system, and unfortunately for you, the new system requires employee records be stored completely differently.</p>&#x000A;&#x000A;<p dir="auto">Your task is to help bridge the gap by creating a Python script able to convert your employee records to the required format. Your script will need to do the following:</p>&#x000A;&#x000A;<ul dir="auto">&#x000A;<li>Import the <code>employee_data1.csv</code> and <code>employee_data2.csv</code> files, which currently holds employee records like the below:</li>&#x000A;</ul>&#x000A;&#x000A;<pre class="code highlight js-syntax-highlight plaintext" v-pre="true"><code>Emp ID,Name,DOB,SSN,State&#x000A;214,Sarah Simpson,1985-12-04,282-01-8166,Florida&#x000A;15,Samantha Lara,1993-09-08,848-80-7526,Colorado&#x000A;411,Stacy Charles,1957-12-20,658-75-8526,Pennsylvania&#x000A;</code></pre>&#x000A;&#x000A;<ul dir="auto">&#x000A;<li>Then convert and export the data to use the following format instead:</li>&#x000A;</ul>&#x000A;&#x000A;<pre class="code highlight js-syntax-highlight plaintext" v-pre="true"><code>Emp ID,First Name,Last Name,DOB,SSN,State&#x000A;214,Sarah,Simpson,12/04/1985,***-**-8166,FL&#x000A;15,Samantha,Lara,09/08/1993,***-**-7526,CO&#x000A;411,Stacy,Charles,12/20/1957,***-**-8526,PA&#x000A;</code></pre>&#x000A;&#x000A;<ul dir="auto">&#x000A;<li>&#x000A;<p>In summary, the required conversions are as follows:</p>&#x000A;&#x000A;<ul>&#x000A;<li>The <code>Name</code> column should be split into separate <code>First Name</code> and <code>Last Name</code> columns.</li>&#x000A;<li>The <code>DOB</code> data should be re-written into <code>DD/MM/YYYY</code> format.</li>&#x000A;<li>The <code>SSN</code> data should be re-written such that the first five numbers are hidden from view.</li>&#x000A;<li>The <code>State</code> data should be re-written as simple two-letter abbreviations.</li>&#x000A;</ul>&#x000A;</li>&#x000A;<li><p>Special Hint: You may find this link to be helpful—<a href="https://gist.github.com/afhaque/29f0f4f37463c447770517a6c17d08f5" rel="nofollow noreferrer" target="_blank">Python Dictionary for State Abbreviations</a>.</p></li>&#x000A;</ul>&#x000A;&#x000A;<h2 dir="auto">&#x000A;<a id="option-4-pyparagraph" class="anchor" challenge, you get to play the role of chief linguist at a local learning academy. As chief linguist, you are responsible for assessing the complexity of various passages of writing, ranging from the sophomoric Twilight novel to the nauseatingly high-minded research article. Having read so many passages, you've since come up with a fairly simple set of metrics for assessing complexity.</p>&#x000A;&#x000A;<p dir="auto">Your task is to create a Python script to automate the analysis of any such passage using these metrics. Your script will need to do the following:</p>&#x000A;&#x000A;<ul dir="auto">&#x000A;<li><p>Import a text file filled with a paragraph of your choosing.</p></li>&#x000A;<li>&#x000A;<p>Assess the passage for each of the following:</p>&#x000A;&#x000A;<ul>&#x000A;<li>Approximate word count</li>&#x000A;<li>Approximate sentence count</li>&#x000A;<li>Approximate letter count (per word)</li>&#x000A;<li>Average sentence length (in words)</li>&#x000A;</ul>&#x000A;</li>&#x000A;<li><p>As an example, this passage:</p></li>&#x000A;</ul>&#x000A;&#x000A;<blockquote dir="auto">&#x000A;<p>“Adam Wayne, the conqueror, with his face flung back and his mane like a lion's, stood with his great sword point upwards, the red raiment of his office flapping around him like the red wings of an archangel. To date, we've learned how to import modules like <code>csv</code>; to read and write files in various formats; to store contents in variables, lists, and dictionaries; to iterate through basic data structures; and to debug along the way. Using what we've learned, try to break down you tasks into discrete mini-objectives. This will be a <em>much</em> better course of action than attempting to Google Search for a miracle.</p></li>&#x000A;<li><p>As you will discover, for some of these challenges, the datasets are quite large.
